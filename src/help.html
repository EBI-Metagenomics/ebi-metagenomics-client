Help

All documentation relating to the use of MGnify was moved to Read the Docs. This allows us to provide a rich, searchable and downloadable document, in addition to different versions of the documentation corresponding to the different pipelines (accessed via the bottom left corner of the EMG Read the Docs site). Answers to many of the most commonly asked questions can be found under the FAQ section. Furthermore, there are also the following training materials as part of the EBI Train online:

  1. Tutorial about the website
  2. Tutorial describing the data submission process
  3. A webinar explaining about MGnify and the analysis you can perform

However, if neither the documentation nor online materials answer your question, please contact us.
