EBI Metagenomics Help

All documentation relating to the use of the EBI metagenomics has been moved to Read the Docs. This allows us to provide a rich, searchable and downloadable document, as well as enables us to store different versions corresponding to the different pipelines (accessed via the bottom left corner of the EMG Read the Docs site). Answers to many of the most commonly asked questions can be found under the FAQ section. Furthermore, there are also the following training materials as part of the EBI Train online:

  1. Tutorial about the website
  2. Tutorial describing the data submission process
  3. A webinar explaining about EBI metagenomics and the analysis you can perform

However, if neither the documentation nor online materials answer your question, please contact us.
