Help

MGnify is a freely available hub for the analysis and exploration of metagenomic, metatranscriptomic, amplicon and assembly data. The resource provides rich functional and taxonomic analyses of user-submitted sequences, as well as analysis of publicly available metagenomic datasets held within the European Nucleotide Archive (ENA).

User Documentation

MGnify user documentation was moved to MGnify Read the Docs site.

This provides a rich, searchable and downloadable overview of the service for scientists and a complete version history corresponding to the different pipelines (accessed via the bottom left corner of the). Answers to many of the most commonly asked questions can be found under the FAQ.

Sequence search

This allows searching our non-redundant protein database using lite version of HMMER. The comprehensive documentation discribing the use of HMMER through sequence search website with all available options, providing background on profile HMMs and the details of the different algorithms is avaiable on HMMER Read the Docs site.
